Equancy vous forme à la data

Publié le 1 sept. 2020

Maîtriser Python, les méthodes de machine learning, l’architecture d’un data lake… Formez-vous à la data avec Equancy ! 

Des formations pour tous, accessibles à tout niveau et en petits groupes

Non pas un, mais deux formateurs pour vous accompagner

Un coaching personnalisé autour de vos problématiques et cas d’usage

Un enseignement collaboratif entre participants, autour d’ateliers pratiques

 

Nos formations débutant :

  • Maîtriser Python pour la data science 
  • Maîtriser les algorithmes de machine learning 
  • Maîtriser pySpark pour vos pipelines de données 
  • Architecturer un data lake dans le Cloud 

 

Nos formations avancées :

Ces formations s’adressent à des data scientists & data engineering ayant un premier niveau de maîtrise de Python pour la data science :

  • Faire ses premiers pas en deep learning (images & textes) 
  • Traiter et prédire les séries temporelles en Python
  • Industrialiser la data science : débuter en MLOps 
  • Créer ses premières applications analytiques avec Dash
